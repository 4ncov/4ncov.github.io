## 4nCoV

**4nCoV**是一个为灾区提供信息服务的开源平台项目。平台为`捐助者`/`赞助商`与`慈善机构`之间搭建桥梁。发布机构物资需求信息，利用互联网平台提供高效支援信息。

在众多机构中，发布需求信息没有统一的平台，意在为全国有需求发功的机构提供发布平台。
平台展示需求信息，提供地区查找、需求物品分类等功能。

目前需要大量的开发者参与进来，需要志愿者们的支持。
